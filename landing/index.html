Features

⚡

Reaction Tipping

React with 💰 on any message to tip the author. Instant. No commands needed.

🔏

Blind Token Privacy

Tips use Cashu blind signatures. The server signs tokens it cannot read. Amounts stay hidden.

🎭

Truly Anonymous Tips

Use /anontip for a Zcash payment link. Send directly from your wallet. The bot sees nothing.

🌧️

Rain Events

Split ZEC across everyone who reacts within 2 hours. Amounts stay private.

🛡️

Shielded Entry & Exit

Zcash shielded addresses for deposits and withdrawals. Amounts hidden on-chain.

🏆

Count-Only Leaderboard

Ranked by number of tips, not amounts. Recognition without financial exposure.
